# BetterStats 测试账号

## 第一阶段开发测试数据

### 快速使用指南

1. **初始化数据库**（首次运行）
```bash
npm run db:init
```

2. **启动服务**
```bash
# 启动后端
npm run server

# 启动前端（新终端）
npm run dev
```

3. **访问系统**
- 前端地址：http://localhost:8000
- 后端 API: http://localhost:17342
